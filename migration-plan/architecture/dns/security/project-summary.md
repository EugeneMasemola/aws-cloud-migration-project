# Project Summary

## Project Name

AWS Website Migration Portfolio

## Background

This portfolio project is based on real cloud migration experience from my time at Didusec. The work involved supporting website migration from traditional hosting environments to AWS and helping with AWS provisioning activities.

## Business Problem

The websites needed to be moved from traditional hosting to a more cloud-ready environment. This required careful migration planning, backup preparation, DNS coordination, hosting setup, and post-migration validation.

## Solution Approach

The migration approach included:

- Reviewing the existing hosting environment
- Preparing AWS hosting resources
- Backing up website files and configuration
- Supporting website transfer activities
- Preparing DNS cutover steps
- Testing website availability after migration
- Documenting the migration process

## My Contribution

My involvement included:

- Supporting AWS infrastructure preparation
- Assisting with migration planning
- Helping with website hosting setup
- Supporting DNS and deployment validation
- Preparing technical documentation
- Contributing to troubleshooting and post-migration checks

## Outcome

The project improved my practical understanding of cloud migration, AWS hosting, DNS planning, infrastructure documentation, and production support.
