# Migration Checklist

## Pre-Migration

- Review current hosting environment
- Identify website files and dependencies
- Confirm DNS provider access
- Confirm backup availability
- Prepare rollback plan
- Prepare AWS hosting environment

## Migration

- Backup website files
- Transfer website files to AWS environment
- Configure hosting settings
- Test website using temporary URL or staging configuration
- Prepare DNS cutover

## Post-Migration

- Validate website availability
- Check internal links and pages
- Confirm DNS propagation
- Monitor for errors
- Document issues and fixes
- Confirm final handover
